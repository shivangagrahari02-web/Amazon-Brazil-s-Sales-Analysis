# Amazon Brazil's Sales Analysis
Analyzed Amazon Brazil's data to identify trends and customer behaviors that could be  leveraged in the Indian market. This comprehensive analysis will help Amazon India enhance customer experience and  seize new market opportunities.

Focused on customer demographics and behavior using the 
Customers table to understand purchase patterns and preferences. Evaluated regional trends 
and customer density through the Geolocation table. Track order lifecycles, product 
preferences, and seller performance using the Orders, Order Items, Product, and Seller 
tables. Additionally, analyzed payment preferences and transaction details via the Payments 
table.
